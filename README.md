# inventoryRiwi
Sistema de gestión CRUD de inventarios desarrollado en Python manejado desde la consola, diseñado bajo una arquitectura modular MVC. El objetivo principal es centralizar la administración de productos, el control de existencias y la persistencia de datos mediante archivos CSV.

# Funcionalidades
- Agregar productos al inventario
- Listar producto(s)
- Buscar producto por nombre
- Calcular estadisticas
- Guardar inventario en un csv
- Cargar inventario desde un CSV
